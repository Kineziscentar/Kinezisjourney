# Kinezis Journey — Online Proof Sprint

Prva prava Next.js verzija spremna za GitHub i Vercel.

## Što radi
- prebacivanje između prikaza člana i trenera
- Coach dashboard
- wizard za kreiranje cilja
- Goal Kickoff za člana
- Today, Ciljevi, Coach i Profil
- osnovne demo interakcije

## Lokalno
```bash
npm install
npm run dev
```

## Vercel
1. Napravi novi GitHub repository.
2. Uploadaj sadržaj ove mape u repository.
3. Na Vercelu odaberi Add New → Project.
4. Importaj GitHub repository.
5. Klikni Deploy. Za ovaj sprint nisu potrebne environment varijable.

## Sljedeće
Nakon potvrde javnog URL-a spajamo postojeći Supabase projekt i dodajemo stvarni login i trajno spremanje ciljeva.
